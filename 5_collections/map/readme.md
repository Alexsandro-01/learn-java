Agenda com `Map`


Um programa que mantém uma agenda de contatos, associando um nome a um telefone.

Nesse programa, a classe se chama AgendaMapa, poder ser instanciada e define os seguintes métodos:

- `void incluirContato(String nome, String telefone)`: que inclui o contato na agenda;
- `void excluirContato(String nome)`: que exclui o contato da agenda;
- `boolean verificaContato(String nome)`: que verifica se o contato está presente na agenda;
- `int tamanhoAgenda()`: que consulta o número atual de contatos da agenda;
- `atualizarTelefone(String nome, String telefone)`: que muda o telefone de um contato;
- `obterTelefone(String nome)`: que obtém o telefone associado a uma pessoa.
